School Registration

All school has to go through registration process on our web App which will done by Efull Technology administrator Staff

The process is as follow

Collect all needed school data

Goto https://sandbox-eschool.efulltech.com.ng/login to log in as super admin

click on schools on the left sidebar to create new

Create a new school with a default admin login credentials

Login

All user can Login on the Web with different access level

Dashboard

Dashboard displays the statistics of the school management activities

Create Parents

Parents are created by the school administrator

The process are as follow

collect the data of the parent

Goto https://sandbox-eschool.efulltech.com.ng/parents

Click on add parent button to create a fresh account or click a Add Account button to add existing users

Fill in all data and press submit button

collect the data of the student

Teachers are created by the school administrator

The process are as follow

collect the data of the teacher

Goto https://sandbox-eschool.efulltech.com.ng/teacher

Click on add parent button to create a fresh account or click a Add Account button to add existing users

Fill in all data and press submit button

Action Buttons

Different actions can be made on any users like updating, deleting, and view details

To view, update or delete any user by the admin just click on any of the action button that best describe your action

All action can done by the admin except changing of password of any users

Create Classes

Classes are created by the school administrator

The process are as follow

collect the data of the parent

Goto https://sandbox-eschool.efulltech.com.ng/classes

Click on add class button to create a new class

Fill in all class details and press submit button

Note that classes are expected to be created prior to adding users to the school database
